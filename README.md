# ARM SCHOOL MANAGEMENT SYSTEM

A simple yet fast and rich school management system licensed under MIT. It comes with ten (10) available modules:

- Teachers
- Students
- Add/Edit Class
- Time Table
- Add/Edit Subjects
- Class List
- Scores
- Class Schedule
- Subject
- Report
